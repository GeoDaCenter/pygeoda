pygeoda 0.0.4
=============

pygeoda is a python library for spatial data analysis based on `GeoDa <https://geodacenter.github.io>`_ and libgeoda, 
which is a C++ library designed for programmers to
do spatial data analysis using their favoriate programming languages. 
It also aims to be easily integratd with other 
libraries, softwares or systems on different platforms.
