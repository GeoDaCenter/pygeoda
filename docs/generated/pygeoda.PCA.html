pygeoda.PCA¶

class pygeoda.PCA(data)[source]¶

Principal component analysis .. note:: The principal components can be interpreted as the best linear approximation to the multivariate point cloud of the data. The goal is to find a small number of principal components,that explains the bulk of the variance in the original variables.

Parameters

data (tuple) – A 2d numeric list with values of selected variables

Returns

A PCA object that wrappers the c++ PCAResult object containing the details of PCA computation

Return type
